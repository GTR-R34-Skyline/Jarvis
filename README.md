# Jarvis

Jarvis is a simple Python-based voice assistant inspired by Iron Manâ€™s JARVIS.
It uses speech recognition and text-to-speech to listen and respond to voice commands.

## Features
- Voice command recognition
- Text-to-speech responses
- Music playback support
- Easy to extend with new commands

## Requirements
- Python 3.7+
- speech_recognition
- pyttsx3
- pyaudio

## Installation
1. Clone the repository
   git clone https://github.com/GTR-R34-Skyline/Jarvis.git

2. Install dependencies
   pip install -r requirements.txt

## Usage
Run the assistant using:
python Jarvis.py

Speak a command when prompted.

## Files
- Jarvis.py : Main assistant script
- music_library.py : Handles music-related commands
